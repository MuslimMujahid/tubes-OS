# tubes-1-os

## Script shell

### Show files and folders in current directory

```shell
ls
```

### Create folder in current directory

```shell
mkdir *foldername*
```

### Run a runable file

```shell
run *filename*
```

### Move file

```shell
mv *origin path* *destination path*
```

### Remove file or folder in current directory

```shell
rm *file/folder*
```

### Print file's content

```shell
cat *file*
```

### Copy file

```shell
cp *filename* *destination path*
```
